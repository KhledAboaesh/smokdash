# SmokeDash ERP - User Guide

## Quick Start Guide

### Login
- Default Username: `admin`
- Default Password: `admin123`

### Main Features
1. **POS (Point of Sale)** - Process sales transactions
2. **Inventory** - Manage products and stock
3. **Customers** - Track customer information and debts
4. **Invoices** - View and print invoices
5. **Reports** - Generate sales and shift reports
6. **Users** - Manage staff and permissions
7. **Settings** - Configure system preferences

### Daily Workflow
1. **Open Shift** - Start your work shift with opening cash
2. **Process Sales** - Use POS to sell products
3. **Close Shift** - End shift and generate Z-Report

### Navigation
- Use sidebar buttons to switch between pages
- Scroll in sidebar if buttons are not visible
- All pages support scrolling for long content

### Keyboard Shortcuts
- `F1` - Open POS
- `F2` - Open Inventory
- `F5` - Refresh data
- `Esc` - Close dialog
